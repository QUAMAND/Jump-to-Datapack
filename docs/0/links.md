# 내부 사이트
- [:compass: .jar 다운로드](jar.html): 설정한 버전의 `client`/`server` `.jar` 파일을 다운로드 할 수 있습니다.

# 외부 사이트
- [:compass: mojira.dev](https://mojira.dev): [마이그레션](https://minecraft.wiki/w/Bug_tracker#Migration) 이후 매우 느려진 일반 [버그 제보 사이트](https://bugs.mojang.com/browse/MC)보다 더 빠르게 버그를 추적하고 검색할 수 있는 사이트입니다.

- [:compass: mcstacker.net](https://mcstacker.net): 많은 명령어를 생성하고, 특히 `nbt` 추가와 수정 그리고 `text_component`에 대한 편집에 편리한 사이트입니다.

- [:compass: misode.github.io](https://misode.github.io): 데이터 팩과 관련한 매우 많은 작업을 할 수 있는 사이트입니다.

- [:compass: mcsrc.dev](https://mcsrc.dev): 소스 코드를 웹으로 볼 수 있는 사이트입니다.

- [:compass: version-diff](https://cccode.pages.dev/version-diff): 특정 버전과의 차이를 시각적으로 표시 해주는 사이트입니다.

- [:compass: bdengine.app](https://bdengine.app): 디스플레이 엔티티로 모델링을 할 수 있는 사이트입니다.
