## enchantment
공식에 사용하는 인챈트입니다.  
이 인챈트를 기준으로 레벨을 계산합니다.

## formula
추가로 지급할 아이템의 양을 계산합니다.

### binomial_with_bonus_count
항상 1개는 제공하며, 추가로 지급할 아이템은 이항 분포로 계산됩니다.
- `extra`: 정수
- `probability`: 확률

- `n` = 인챈트 레벨 + `extra`

# enchant_randomly
해당 아이템에 무작위 인챈트를 추가합니다.

## options
어떤 인챈트가 선택될 것인지 설정합니다.  
비어 있을 경우, 무작위적인 인챈트가 부여됩니다.

- `id`: 단일 인챈트를 지정합니다.
   - `minecraft:fortune`
- `tag`: 여러 개로 엮인 인챈트들을 지정합니다.
   - `#minecraft:weapon_enchantments`
- `list`: 인챈트 목록을 직접 지정합니다.
   - `["minecraft:sharpness", "minecraft:smite"]`

## only_compatible
해당 인챈트가 아이템에 적용되는지 여부를 설정합니다.
- `true`: 예를 들어, 검 종류의 아이템에는 효율(`minecraft:efficiency`)을 부여할 수 없습니다.
   - 단, 책(`minecraft:book`)은 항상 `false`입니다.
- `false`: 예를 들어, 검 종류의 아이템에는 효율(`minecraft:efficiency`)을 부여할 수 있습니다.

# enchant_with_levels
마법 부여대에서 인챈트를 부여하는 것과 같이 아이템에 인챈트를 부여합니다.  
`enchant_randomly`와는 다르게 부여되는 인챈트의 종류가 고정되어 있습니다.

## levels
마법 부여대에서 플레이어의 레벨을 적용하는 것과 같습니다.
- `"levels": 30`: 플레이어 30레벨과 동일합니다.

## options
어떤 인챈트가 선택될 것인지 설정합니다.  
비어 있을 경우, 무작위적인 인챈트가 부여됩니다.

- `id`: 단일 인챈트를 지정합니다.
   - `minecraft:fortune`
- `tag`: 여러 개로 엮인 인챈트들을 지정합니다.
   - `#minecraft:weapon_enchantments`
- `list`: 인챈트 목록을 직접 지정합니다.
   - `["minecraft:sharpness", "minecraft:smite"]`

# limit_count
아이템의 개수를 제한합니다.

## limit
- `int`: 이 값으로 설정합니다.
- `object`: `min` ~ `max` 값으로 설정합니다.
   - `type`: [[number-provider](https://minecraft.wiki/w/Loot_table#Number_provider)]를 참고하십시오.

```json
{
   "function": "minecraft:limit_count",
   "limit": {
     "min": {},
     "max": 5
   }
}
```
